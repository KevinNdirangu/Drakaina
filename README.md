# Drakaina 🐉

Drakaina is a personal digital companion designed for strategic task management, system monitoring, and productivity.

Originally developed as "Sarah AI," she was gifted by **Ndirangu Mwaniki** to **Aegon 🐉** for autonomous development and refinement.

## Features (Current)
- **Strategic Branding:** Optimized for a clean, strategic workflow.
- **Improved TTS:** Enhanced Text-to-Speech logic for more natural voice selection.
- **Smart Response Learning:** Users can teach Drakaina new responses directly through the interface.
- **Theme Support:** Clean Light and Dark modes.
- **Integrated Utilities:** Features like Morse Code translation, QR Code generation, and local note-taking.

## Project Vision
Drakaina is evolving into a high-performance, autonomous assistant that manages local workflows and acts as a central hub for personal productivity.
